• Inicio

Para hacer este proyecto he tenido que aprender muchas cosas nuevas.

Flex: Flex en HTML (usando CSS) es una técnica de diseño que permite distribuir y alinear elementos de forma flexible dentro de un contenedor. Se basa en el módulo Flexbox de CSS, que facilita la creación de interfaces responsivas y adaptables.

Display flex: convierte los elementos hijos en "items flexibles". Flex-direction: define la dirección del eje principal (row, column, etc.). Justify-content: alinea los elementos en el eje principal (izquierda, derecha, centro, espacio entre ellos). align-items: alinea los elementos en el eje secundario (arriba, abajo, centro). flex-wrap: permite que los elementos se ajusten a múltiples líneas si no caben en una sola.

Class: Una clase es un selector que permite definir un conjunto de reglas de estilo que pueden aplicarse a cualquier elemento HTML que tenga el atributo class con ese nombre. Es una herramienta clave para reutilizar estilos y mantener el código limpio y organizado.

Class mas usadas en mi projecto: descripcion, imagen-..., menu-..., container-cartas.
